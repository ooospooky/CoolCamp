# CoolCamp
此作品為大四專題，所有前端項目接由我開發




### 綱要

  我們希望能讓喜愛露營或者初次想嘗試露營的人，能夠快速又方便的找到適合自己的營地。
更重要的是能讓露營客避免一些沒有用心經營或是骯髒、不舒服的環境。


### 內容說明

讓使用者清楚看到營地評分且能篩選不同位置的露營地，且能為露營地評分，並留下對此營地的想法或建議。

### 使用技術

-JS/HTML/CSS

-React

-Bootstrap/SemanticUI

-Google OAuth2.0

-MapBox

-JsonServer 

-RESTful API
